Convert excel/.csv file to laravel PHP databases
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
HOW TO RUN:

pre requisites:

uncomment extension=zip from your php.ini file , ie, 
;extension=zip to extension=zip

1. php artisan migrate (for creating database)
2. php artisan serve
